---
title: Viewing Conversations
description: How to work with the conversation list, filters, tags, and search in the monitoring section
---

The **Chats** section contains all conversations between your AI agent and customers. The conversation list updates in real time — you see new messages instantly, without reloading the page.

## Conversation list

When you open the monitoring section, you will see a list of all conversations sorted by the date of the last message. Each conversation displays:

- **Customer name** or channel identifier
- **Last message preview**
- **Time** of the last message
- **Channel source** — WhatsApp, Instagram, Telegram, etc.
- **Current mode** — AI Mode or Operator Mode
- **Read/unread status** — unread conversations are visually highlighted

## Filtering by channel

Use the channel source filter to show only the conversations you need:

- WhatsApp
- Instagram
- Telegram
- Other connected channels

The filter helps you quickly switch between platforms when you have multiple channels connected.

## Tagging system

You can assign tags to conversations for organization and classification:

- Create your own tags (e.g., "VIP", "Return", "Awaiting response")
- Filter the conversation list by tags
- A single conversation can have multiple tags

Tags help your team navigate the flow of conversations more quickly.

## Message search

The search bar lets you find conversations by message content. Enter a keyword or phrase, and the system will show conversations containing matches.

## Viewing a conversation

Click on any conversation to open the full message history. You will see:

- All customer messages and AI agent responses
- The time each message was sent
- Attached files, images, and voice messages
- Moments when the conversation switched between AI Mode and Operator Mode
- Feedback buttons (like/dislike) next to AI agent messages

## Message feedback

Next to each AI agent response, there are like/dislike buttons. If a response is inaccurate, click dislike — a correction field will appear. This feedback is used to improve the AI agent.

## Notifications

The notification sound toggle lets you enable or disable audio alerts for new messages. This is convenient if you are constantly working in the monitoring dashboard.

<Callout title="Tip">
  Pay attention to unread conversations and messages with dislikes — this signals that you need to check the response quality or update the knowledge base.
</Callout>
