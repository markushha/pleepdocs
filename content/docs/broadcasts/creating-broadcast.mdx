title: Создание рассылки

---

Создание рассылки в Pleep — это простой процесс из нескольких шагов. Перейдите в **Главная** → **Рассылки** и выберите нужный тип рассылки на соответствующей вкладке.

## Выбор типа рассылки

Перед созданием определите, какой тип рассылки вам нужен:

- **WhatsApp Broadcast** — быстрая отправка сообщений нескольким контактам
- **WABA Broadcast** — рассылка через шаблоны, одобренные Meta (основной способ)
- **WABA Coexistence Broadcast** — гибридный режим AI-агента и живого оператора
- **Voice Broadcast** — автоматические голосовые звонки через голосового AI-агента

<Callout title="Перед началом">
  Для WABA-рассылок убедитесь, что у вас есть хотя бы один одобренный шаблон сообщения. Если шаблона нет — сначала [создайте его](/docs/broadcasts/templates).
</Callout>

<Steps>

### Выберите аудиторию

Определите, кому вы хотите отправить рассылку:

- **Все контакты** — отправка всем клиентам из вашей базы
- **Сегмент** — выберите группу контактов по определённым критериям (например, город, дата последнего обращения)
- **Список вручную** — загрузите список номеров телефонов

Используйте фильтрацию получателей, чтобы точно определить целевую аудиторию.

### Выберите шаблон (для WABA)

Для WABA-рассылок выберите один из одобренных шаблонов сообщений. Вы увидите предварительный просмотр того, как будет выглядеть сообщение.

### Настройте переменные

Если шаблон содержит переменные (например, `{{1}}` — имя клиента), укажите, какие данные подставлять:

- **Имя клиента** — будет подставлено из карточки контакта
- **Произвольный текст** — одинаковый для всех получателей
- **Поле контакта** — любое поле из карточки клиента

### Настройте время отправки

Выберите, когда отправить рассылку:

- **Немедленно** — отправка начнётся сразу после подтверждения
- **Запланировать** — выберите дату и время для отложенной отправки

### Предварительный просмотр

Проверьте, как будет выглядеть сообщение для конкретного получателя:

- Убедитесь, что переменные подставлены корректно
- Проверьте текст на ошибки
- Убедитесь, что ссылки и кнопки работают правильно

### Отправьте рассылку

Нажмите **Отправить** для подтверждения. Pleep начнёт доставку сообщений, и вы сможете отслеживать прогресс в реальном времени.

</Steps>

## После отправки

- Отслеживайте статус доставки каждого сообщения в разделе рассылок
- При неудачной доставке система автоматически повторит попытку
- Ответы клиентов будут обрабатываться AI-агентом автоматически (или менеджером в режиме Coexistence)
- Вы получите уведомление, если клиент запросит связь с менеджером

<Callout title="Обратите внимание">
  Meta взимает плату за каждое доставленное шаблонное сообщение (WABA). Убедитесь, что у вас настроена [оплата](/docs/broadcasts/payment).
</Callout>
