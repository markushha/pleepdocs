title: Feedback from Chats
description: How likes and dislikes in monitoring relate to AI agent improvement
---

In the monitoring section, next to each AI agent response there are feedback buttons — like and dislike. This system lets you quickly rate response quality while reviewing conversations.

## How feedback works

### Like

Click like if the AI agent's response is correct and helpful. This helps the system understand which responses are considered good.

### Dislike

Click dislike if the response is inaccurate, incomplete, or inappropriate. After clicking, a correction field will appear where you can:

- Describe what exactly is wrong
- Provide the correct answer
- Add context

## Connection to improvement

Feedback from chats complements the [Improvement](/docs/improvement/how-it-works) interface. You can:

1. Notice a problematic response in monitoring and dislike it with a description
2. Go to the improvement section and describe the problem in more detail
3. The system will update the agent settings

Regular feedback helps identify systemic issues — if the same error repeats, it signals the need for a deeper correction.

## What to include in a correction

To make a correction useful, include:

- **What is wrong** — the specific problem with the response (incorrect information, wrong tone, unnecessary details)
- **What it should be** — the correct version of the response
- **Context** — why the current response is inappropriate

## Example of good correction

> "A customer asked if the store is open on Sundays. The AI agent said 'yes, from 10:00 to 20:00', but we are closed on Sundays. The working hours information needs to be updated."

<Callout title="Every piece of feedback matters">
  Do not skip inaccurate responses — even small corrections help the AI agent get better. Use dislike every time you see a problem.
</Callout>
