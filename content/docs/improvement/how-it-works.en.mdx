AI agent improvement in Pleep happens through a chat-like interface in the **Improvement** section. You describe the problem in plain language, and the system automatically updates the agent's settings.

## The improvement process

<Steps>

### Describe the problem

Open the improvement interface and write what needs to be fixed. Use natural language — no technical knowledge required. For example:

- "The agent responds with messages that are too long, I need shorter answers"
- "When asked about the price of product X, the agent gives the wrong amount"
- "The agent does not ask clarifying questions before making a recommendation"

### The system validates your input

Before processing, the system checks your request for specificity. If the description is too vague (e.g., "make it better" or "fix the responses"), the system will ask you to clarify what exactly needs to change.

### AI generates updated instructions

Based on your description, the system analyzes the current agent settings and generates an updated version with the changes applied.

### Review changes

The system shows a summary of the changes made — what was added, modified, or removed from the agent settings.

### Save and history

Changes are saved automatically. The full correction history is available in the interface — you can see what improvements were made previously.

</Steps>

## Templates for common problems

The improvement interface provides pre-defined templates for frequent corrections:

- **Brevity** — make the AI agent's responses shorter and more to the point
- **Pricing** — fix behavior when customers ask about prices
- **Question handling** — change how the AI agent reacts to certain types of questions

Templates are a quick way to apply standard improvements without needing to describe the problem in detail.

## What happens behind the scenes

The system does not simply insert your text. It analyzes the current agent settings, determines which sections need to change, and generates an updated version while keeping the remaining settings intact.

<Callout title="Please note">
  Changes are applied to agent settings immediately. If you are not satisfied with the result, you can make another correction through the same chat interface.
</Callout>
