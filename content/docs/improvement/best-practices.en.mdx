title: Best Practices
description: Tips for effectively improving your AI agent through the coaching interface
---

To get the most out of the improvement interface, follow these recommendations.

## Be specific

The system validates your input for specificity and rejects vague requests. The more precise your description, the more accurate the changes to agent settings will be.

**System will reject:** "Make the responses better"

**System will accept:** "When a customer asks about delivery to Astana, the agent should respond: delivery takes 3—5 business days and costs 2,000 tenge"

## Use templates

For common corrections, use the pre-defined templates in the improvement interface:

- **Brevity** — if the agent responds with messages that are too long
- **Pricing** — if the agent handles prices incorrectly
- **Question handling** — if you need to change the response logic for certain question types

Templates save time and contain proven formulations.

## One problem at a time

Describe one problem per message. This allows the system to more precisely determine which instruction sections need to change and simplifies tracking the correction history.

## Verify results in the sandbox

After each improvement:

1. Open the [Sandbox](/docs/sandbox)
2. Ask the same question that caused the issue
3. Verify that the response is now correct
4. Try similar phrasings — make sure the fix works across different variations

## Review the change history

The improvement interface saves the full correction history. Periodically review it to:

- Make sure previous improvements do not conflict with new ones
- Understand which problems occur most frequently
- Track the evolution of agent settings

## Keep the knowledge base up to date

Not all problems can be solved by improving instructions. If the agent provides incorrect factual information (prices, timelines, availability), check and update the [knowledge base](/docs/knowledge-base).

<Callout title="Golden rule">
  Specific problem description + sandbox verification = effective AI agent improvement.
</Callout>
