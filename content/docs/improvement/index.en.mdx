The **Improvement** section is a chat-like interface for coaching your AI agent. Instead of manually editing settings, you describe the problem in plain language and the system automatically updates the agent settings.

## How it works

The Improvement section works like a chat. You write what needs to be fixed, and the system:

1. Validates your input for specificity (rejects vague descriptions)
2. Generates updated agent settings
3. Shows a summary of the changes made
4. Maintains a history of corrections

No technical skills needed — just describe the problem.

## Improvement tools

- **[How Improvement Works](/docs/improvement/how-it-works)** — the detailed process from describing a problem to updating instructions
- **[Best Practices](/docs/improvement/best-practices)** — tips for effectively describing problems and using templates
- **[Feedback from Chats](/docs/improvement/feedback)** — how likes/dislikes in monitoring relate to agent improvement

<Callout title="Tip">
  Use pre-defined templates for common fixes — brevity, pricing, question handling. Templates are available directly in the improvement interface.
</Callout>
