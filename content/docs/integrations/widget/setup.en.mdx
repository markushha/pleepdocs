---
title: Widget Setup
description: Step-by-step guide to installing the Pleep chat widget on your website.
---

Follow these steps to add the Pleep chat widget to your website.

## Step 1. Open Widget Settings

1. Log in to the Pleep dashboard.
2. Go to **Integrations** → **Widget**.

## Step 2. Customize Appearance

Style the widget to match your website:

- **Colors** — choose a primary color that matches your brand.
- **Position** — select which corner of the screen the chat button will appear in (typically bottom-right).
- **Greeting message** — set the text visitors will see when they open the chat.

## Step 3. Copy the Embed Code

After customizing, click **Copy Code**. You will receive an HTML/JavaScript snippet to add to your site.

## Step 4. Paste the Code into Your Website

Insert the copied code into your website's HTML before the closing `</body>` tag:

```html
<!-- Pleep Widget -->
<script src="https://widget.pleep.app/embed.js" async></script>
<!-- Paste before </body> -->
```

> If you use a CMS (WordPress, Tilda, etc.), add the code through the custom HTML injection settings.

## Step 5. Test the Widget

1. Open your website in a browser.
2. Verify that the chat button appears in the specified position.
3. Click the button and send a test message.
4. Confirm that the AI agent responds correctly.

If the widget does not appear, make sure the code is inserted correctly and check for errors in the browser console.
