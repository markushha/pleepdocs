The Pleep web widget is an embeddable chat that lets your website visitors talk to the AI agent directly on the page.

## How It Works

The widget appears as a button in the corner of the screen. When a visitor clicks it, a chat window opens where the AI agent is ready to answer questions, help with product or service selection, and collect contact details.

## Features

- Customizable appearance (colors, position, greeting message)
- Automatic responses from the AI agent
- Visitor contact information collection
- Works on any device — desktop, tablet, mobile
- Handoff to a live agent

## Getting Started

Go to the [Widget Setup](/docs/integrations/widget/setup) guide to add the widget to your website.
