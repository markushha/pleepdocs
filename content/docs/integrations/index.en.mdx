Pleep connects to popular messengers and communication channels so your AI agent can engage with customers wherever they prefer.

## Available channels

### WhatsApp (recommended)

The primary channel for customer communication. Pleep supports several WhatsApp connection methods:

- **WABA Coexistence** — the recommended option. A hybrid mode where the AI agent and a live operator can work simultaneously
- **WABA (WhatsApp Business API)** — the official API for sending template messages and broadcasts
- **Official API** — direct connection via the official WhatsApp API
- **Green API** — an alternative provider for WhatsApp integration
- **Wazzup** — integration via the Wazzup service

### Instagram

Integration with Instagram Direct via Facebook/Meta. Lets you receive and process messages from customers who write to you on Instagram. The AI agent responds to incoming messages automatically.

### Telegram

Connection via a Telegram bot. Create a bot through @BotFather, connect it to Pleep, and the AI agent will start responding to messages in Telegram.

### Web Widget

An embeddable chat widget for your website with customizable appearance. Visitors can start a conversation with the AI agent directly on the page without leaving your site.
