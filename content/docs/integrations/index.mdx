Pleep подключается к популярным мессенджерам и каналам связи, чтобы ваш AI-агент мог общаться с клиентами там, где им удобно.

## Доступные каналы

### WhatsApp (рекомендуемый)

Основной канал для работы с клиентами. Pleep поддерживает несколько способов подключения WhatsApp:

- **WABA Coexistence** — рекомендуемый вариант. Гибридный режим, в котором AI-агент и живой оператор могут работать одновременно
- **WABA (WhatsApp Business API)** — официальный API для отправки шаблонных сообщений и рассылок
- **Official API** — прямое подключение через официальный WhatsApp API
- **Green API** — альтернативный провайдер для подключения WhatsApp
- **Wazzup** — интеграция через сервис Wazzup

### Instagram

Интеграция с Instagram Direct через Facebook/Meta. Позволяет принимать и обрабатывать сообщения от клиентов, которые пишут вам в Instagram. AI-агент отвечает на входящие сообщения автоматически.

### Telegram

Подключение через Telegram-бота. Создайте бота через @BotFather, подключите его к Pleep, и AI-агент начнёт отвечать на сообщения в Telegram.

### Веб-виджет

Встраиваемый чат-виджет для вашего сайта с настраиваемым внешним видом. Посетители могут начать диалог с AI-агентом прямо на странице, не покидая сайт.
