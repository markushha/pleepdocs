Pleep lets you connect a Telegram bot so your AI agent can automatically respond to customer messages in Telegram.

## How It Works

1. You create a Telegram bot via @BotFather.
2. You connect the bot to Pleep by providing the token.
3. Customers message the bot in Telegram — the AI agent processes messages and responds automatically.

## Features

- Automatic responses to incoming messages
- Text message support
- Handoff to a live agent when needed
- 24/7 availability

## Getting Started

Go to the [Connecting Telegram](/docs/integrations/telegram/connection-guide) guide to set up the integration.
