The Pleep Voice Agent enables your AI agent to make and receive phone calls, providing a voice channel for communicating with customers.

## Features

- **Voice model selection** — use ElevenLabs voices for natural-sounding speech.
- **Voice customization** — adjust stability, similarity, and speech speed (from 0.5x to 1.5x).
- **First message** — set a greeting phrase that the AI agent uses to start each conversation.
- **Agent script** — define the AI agent's behavior during voice interactions.
- **Incoming call routing** — route inbound calls to the AI agent.
- **Outgoing voice broadcasts** — automated calls to customers for confirmations, reminders, or announcements.
- **Phone number assignment** — assign a phone number to the voice agent.

## Management

In the voice agent section, you can:

- Create a new voice agent.
- Delete an existing voice agent.
- Configure all voice and behavior settings.
