Follow this guide to create and configure the Pleep Voice Agent.

<Callout type="info">
The Voice Agent is available on the PRO plan and above. This feature is in beta.
</Callout>

## Prerequisites

- An active Pleep subscription on the **PRO** plan or higher.
- A configured AI agent with a knowledge base.

## Creating a voice agent

1. Go to the **Voice Agent** section in your application dashboard.
2. Click **Create Voice Agent**.

## Voice configuration

### Voice model selection

Choose from the available ElevenLabs voices. Each voice has unique timbre and intonation characteristics.

### Voice parameters

Customize how the voice agent sounds:

- **Stability** — determines how predictable and even the voice sounds. Higher values produce more stable speech.
- **Similarity** — how closely the voice matches the original model.
- **Speech speed** — adjustable from 0.5x (slow speech) to 1.5x (fast speech).

## Behavior configuration

### First message

Set the phrase the AI agent uses to start each conversation. For example: "Hello! How can I help you?"

### Agent script

Write instructions for the AI agent that define its behavior during voice interactions: communication style, topics, restrictions, and conversation scenarios.

## Call configuration

### Incoming calls

Set up inbound call routing:

1. Assign a phone number to the voice agent.
2. Define routing rules — all calls go to the AI agent, or only under specific conditions.

### Outgoing voice broadcasts

Set up automated outbound calls:

1. Upload a list of numbers to call.
2. Define the conversation scenario.
3. Launch the broadcast.

## Deleting a voice agent

If you need to delete a voice agent, go to its settings and click **Delete**. This action is irreversible.
