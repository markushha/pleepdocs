Pleep ашық API-сі Pleep хабарлама жіберу мүмкіндіктерін өз жүйелеріңізге біріктіруге мүмкіндік береді. WhatsApp үлгі хабарламаларын бір немесе бірнеше алушыға тікелей бэкендіңізден, CRM-нен немесе автоматтандыру құралдарынан жіберуге болады.

## API арқылы не істеуге болады

- **Үлгі хабарламаларды жіберу** — жеке кабинетке кірмей-ақ WhatsApp хабарламаларын өз жүйелеріңізден жіберу
- **Жаппай жіберу** — бір сұранымда бірнеше алушыға жіберу
- **Екі WABA арнасын қолдау** — WABA Coexistence (Meta Cloud API) және ескі WABA (GupShup) арналарымен жұмыс істейді

## Бастау

1. Жеке кабинетте [API кілтін жасаңыз](/kk/api/api-keys)
2. [Үлгі хабарламаны жіберу](/kk/api/send-template) эндпоинт құжаттамасын оқыңыз
3. Алғашқы API сұранымыңызды орындаңыз

<Callout title="API нұсқасы">
  Қазіргі API нұсқасы — **v1**. Барлық эндпоинттердің префиксі `/api/v1/`.
</Callout>

## Аутентификация

Барлық API сұранымдары `X-API-Key` тақырыбында API кілтін қажет етеді. API кілттерін [жеке кабинет → Әзірлеушілерге](https://app.pleep.app/dashboard?cabinet=developers) бөлімінде жасауға және басқаруға болады.

```
X-API-Key: сіздің-api-кілтіңіз
```

<Callout title="API кілтін қауіпсіз сақтаңыз">
  API кілті тек жасалған кезде бір рет көрсетіледі. Оны сенімді жерде сақтаңыз. Жоғалтсаңыз, ескі кілтті қайтарып алып, жаңасын жасаңыз.
</Callout>

## Сұраным лимиттері

API бір API кілтіне **минутына 30 сұраным** рұқсат етеді. Лимиттен асқанда `RATE_LIMITED` кодымен `429` қатесін аласыз. Біраз күтіп, сұранымды қайталаңыз.

## Негізгі URL

Барлық API сұранымдары Pleep негізгі URL-іне жіберіледі:

```
https://microservice.pleep.app/api/v1/
```
