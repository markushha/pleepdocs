---

API кілттері Pleep ашық API-сіне сұранымдарды аутентификациялау үшін қолданылады. API кілттерін [жеке кабинет → Әзірлеушілерге](https://app.pleep.app/dashboard?cabinet=developers) бөлімінде басқаруға болады.

## API кілтін жасау

<Steps>
<Step>

### Жеке кабинетті ашыңыз

Дашбордта жоғарғы оң бұрыштағы профиль белгішесін басып, **Жеке кабинет** тармағын таңдаңыз.

![Жеке кабинет](/images/api/personal-cabinet.png)

</Step>
<Step>

### «Әзірлеушілерге» бөліміне өтіңіз

Жеке кабинеттің сол жақ мәзірінде **Әзірлеушілерге** тармағын таңдаңыз.

![Әзірлеушілерге бөлімі](/images/api/developers-section.png)

</Step>
<Step>

### Кілтті жасаңыз

**API кілтін жасау** батырмасын басыңыз, атауын енгізіңіз (мысалы, «CRM интеграциясы») және кілтті көшіріңіз.

</Step>
</Steps>

<Callout title="Маңызды">
  Толық API кілті тек жасалған сәтте көрсетіледі. Жоғалтсаңыз, ескі кілтті жойып, жаңасын жасаңыз.
</Callout>

## API кілттерін көру

Барлық белсенді API кілттері жеке кабинеттің **Әзірлеушілерге** бөлімінде көрсетіледі. Әр жазбада кілт аты, маскаланған кілт нұсқасы, жасалған күні және соңғы пайдаланылған күні көрсетіледі.

## API кілтін жою

Егер API кілті бұзылған немесе қажет емес болса:

1. [Жеке кабинет → Әзірлеушілерге](https://app.pleep.app/dashboard?cabinet=developers) бөлімін ашыңыз
2. Жойғыңыз келетін кілтті табыңыз
3. Кілттің оң жағындағы жою белгішесін басыңыз

Кілт бірден жойылады. Жойылған кілтпен жасалған кез келген сұраным `401 UNAUTHORIZED` қатесін алады.
