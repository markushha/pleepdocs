---
title: API кілттері
description: Pleep ашық API үшін API кілттерін жасау, көру және қайтарып алу
---

API кілттері Pleep ашық API-сіне сұранымдарды аутентификациялау үшін қолданылады. Кілттерді жеке кабинеттегі **Баптаулар** → **API кілттері** бөлімінде немесе `/api-keys` бетіне тікелей өтіп басқаруға болады.

## API кілтін жасау

1. Жеке кабинетте **Баптаулар** → **API кілттері** бөліміне өтіңіз
2. **API кілтін жасау** батырмасын басыңыз
3. Кілтке сипаттамалық ат беріңіз (мысалы, «CRM интеграциясы» немесе «Тапсырыс хабарландырулары»)
4. Кілтті бірден көшіріңіз — ол тек бір рет көрсетіледі

<Callout title="Маңызды">
  Толық API кілті тек жасалған сәтте көрсетіледі. Жоғалтсаңыз, ескі кілтті қайтарып алып, жаңасын жасау қажет.
</Callout>

## API кілттерін көру

Барлық белсенді API кілттерін жеке кабинетте көруге болады. Әр жазбада кілт аты, маскаланған кілт нұсқасы және жасалған күні көрсетіледі.

## API кілтін қайтарып алу

Егер API кілті бұзылған немесе қажет емес болса:

1. **Баптаулар** → **API кілттері** бөліміне өтіңіз
2. Қайтарып алғыңыз келетін кілтті табыңыз
3. **Қайтарып алу** батырмасын басыңыз

Кілт бірден өшіріледі. Қайтарып алынған кілтпен жасалған кез келген сұраным `401 UNAUTHORIZED` қатесін алады.

## Кілттерді басқару эндпоинттері

Бұл эндпоинттер кілттерді бағдарламалық түрде басқару үшін қолжетімді. Олар API кілтін емес, JWT аутентификациясын (жеке кабинетке кіру сессиясын) қажет етеді.

| Метод | Эндпоинт | Сипаттама |
|-------|----------|-----------|
| `POST` | `/api-keys` | Жаңа API кілтін жасау. Ашық мәтін кілтін бір рет қайтарады |
| `GET` | `/api-keys` | Аккаунтыңыздың барлық API кілттерін тізімдеу |
| `DELETE` | `/api-keys/:id` | Белгілі бір API кілтін қайтарып алу |

<Callout title="Ескертпе">
  Кілттерді басқару эндпоинттері API кілт аутентификациясын емес, жеке кабинетке кіру сессиясын (JWT) пайдаланады. Әдетте кілттерді осы эндпоинттер арқылы емес, жеке кабинет интерфейсі арқылы басқарады.
</Callout>
