---
title: API-ключи
description: Как создать, просмотреть и отозвать API-ключи для публичного API Pleep
---

API-ключи используются для аутентификации запросов к публичному API Pleep. Управлять ключами можно в [личном кабинете → Разработчикам](https://app.pleep.app/dashboard?cabinet=developers).

## Создание API-ключа

<Steps>
<Step>

### Откройте личный кабинет

В дашборде нажмите на иконку профиля в правом верхнем углу и выберите **Личный кабинет**.

![Личный кабинет](/images/api/personal-cabinet.png)

</Step>
<Step>

### Перейдите в раздел «Разработчикам»

В левом меню личного кабинета выберите **Разработчикам**.

![Раздел Разработчикам](/images/api/developers-section.png)

</Step>
<Step>

### Создайте ключ

Нажмите **Создать API-ключ**, введите название (например, «Интеграция с CRM») и скопируйте ключ.

</Step>
</Steps>

<Callout title="Важно">
  Полный API-ключ отображается только в момент создания. Если вы его потеряете, удалите старый ключ и создайте новый.
</Callout>

## Просмотр API-ключей

Все активные API-ключи отображаются в разделе **Разработчикам** личного кабинета. Каждая запись содержит имя ключа, маскированную версию ключа, дату создания и дату последнего использования.

## Удаление API-ключа

Если API-ключ скомпрометирован или больше не нужен:

1. Откройте [личный кабинет → Разработчикам](https://app.pleep.app/dashboard?cabinet=developers)
2. Найдите нужный ключ
3. Нажмите на иконку удаления справа от ключа

Ключ удаляется немедленно. Любые запросы с удалённым ключом получат ошибку `401 UNAUTHORIZED`.
