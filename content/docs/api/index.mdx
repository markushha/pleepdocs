Публичный API Pleep позволяет интегрировать возможности рассылки Pleep в ваши собственные системы. Вы можете отправлять шаблонные сообщения WhatsApp одному или нескольким получателям прямо из вашего бэкенда, CRM или инструментов автоматизации.

## Что можно делать через API

- **Отправлять шаблонные сообщения** — инициировать отправку WhatsApp-сообщений из ваших систем без входа в личный кабинет
- **Массовая отправка** — отправлять нескольким получателям в одном запросе
- **Поддержка обоих каналов WABA** — работает с WABA Coexistence (Meta Cloud API) и устаревшим WABA (GupShup)

## Начало работы

1. [Создайте API-ключ](/docs/api/api-keys) в личном кабинете
2. Изучите документацию эндпоинта [Отправка шаблонного сообщения](/docs/api/send-template)
3. Выполните ваш первый API-запрос

<Callout title="Версия API">
  Текущая версия API — **v1**. Все эндпоинты имеют префикс `/api/v1/`.
</Callout>

## Аутентификация

Все API-запросы требуют API-ключ, передаваемый в заголовке `X-API-Key`. Создать и управлять API-ключами можно в личном кабинете в разделе **Настройки** → **API-ключи**, либо на странице `/api-keys`.

```
X-API-Key: ваш-api-ключ
```

<Callout title="Храните API-ключ в безопасности">
  API-ключ отображается только один раз при создании. Сохраните его в надёжном месте. Если вы его потеряете, отзовите старый ключ и создайте новый.
</Callout>

## Лимиты запросов

API допускает **30 запросов в минуту** на один API-ключ. При превышении лимита вы получите ошибку `429` с кодом `RATE_LIMITED`. Подождите и повторите запрос через некоторое время.

## Базовый URL

Все API-запросы отправляются на базовый URL вашего инстанса Pleep:

```
https://app.pleep.app/api/v1/
```
