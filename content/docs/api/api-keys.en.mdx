---
title: API Keys
description: How to create, view, and delete API keys for the Pleep Public API
---

API keys are used to authenticate requests to the Pleep Public API. You can manage your API keys in the [personal cabinet → Developers](https://app.pleep.app/dashboard?cabinet=developers).

## Creating an API key

<Steps>
<Step>

### Open your personal cabinet

In the dashboard, click the profile icon in the top right corner and select **Personal Cabinet**.

![Personal Cabinet](/images/api/personal-cabinet.png)

</Step>
<Step>

### Go to the Developers section

In the left menu of the personal cabinet, select **Developers**.

![Developers section](/images/api/developers-section.png)

</Step>
<Step>

### Create a key

Click **Create API Key**, enter a name (e.g. "CRM integration"), and copy the key.

</Step>
</Steps>

<Callout title="Important">
  The full API key is displayed only at the moment of creation. If you lose it, delete the old key and create a new one.
</Callout>

## Viewing API keys

All active API keys are displayed in the **Developers** section of your personal cabinet. Each entry shows the key name, a masked version of the key, creation date, and last used date.

## Deleting an API key

If an API key is compromised or no longer needed:

1. Open [personal cabinet → Developers](https://app.pleep.app/dashboard?cabinet=developers)
2. Find the key you want to delete
3. Click the delete icon to the right of the key

The key is removed immediately. Any requests using the deleted key will receive a `401 UNAUTHORIZED` error.
