The **Sandbox** is an isolated environment for testing your AI agent. The interface mirrors the chat from the monitoring section, but all conversations happen in a safe space without affecting real customers.

## Why use the sandbox?

- **Risk-free testing** — verify AI agent responses without affecting real customer conversations
- **Verify changes** — after updating the knowledge base or agent settings, confirm the agent responds correctly
- **File testing** — upload files and images to check how the AI agent processes them
- **Team training** — show colleagues how the AI agent works without any risk to production

## What is available in the sandbox

The sandbox fully replicates the AI agent's behavior:

- The same chat interface as in monitoring
- Full conversation history within the test session
- File and image upload
- Access to the knowledge base and agent settings
- All configured extensions apply

No actions in the sandbox affect real customers or production data.
