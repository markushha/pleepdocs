Расширения позволяют подключить к Pleep внешние сервисы и бизнес-инструменты, значительно расширяя возможности вашего AI-агента.

## Что дают расширения?

Без расширений AI-агент отвечает на вопросы клиентов, используя информацию из базы знаний. С расширениями он может:

- **Работать с CRM** — автоматически создавать лиды, синхронизировать контакты и управлять сделками в AmoCRM или Битрикс24.
- **Управлять записями** — проверять доступные слоты и записывать клиентов через Google Calendar или Altegio.
- **Проверять наличие товаров** — получать актуальную информацию о наличии, ценах и обрабатывать заказы из МойСклад.

## Доступные расширения

| Расширение | Назначение |
|---|---|
| [Google Calendar](/docs/extensions/google-calendar) | Проверка доступности, создание событий из диалогов |
| [AmoCRM](/docs/extensions/amocrm) | Лиды, контакты, воронки, сделки, автоматизация статусов |
| [Битрикс24](/docs/extensions/bitrix24) | CRM-подключение, контакты, сделки, таймлайн |
| [Altegio](/docs/extensions/altegio) | Онлайн-запись, расписание, управление слотами |
| [МойСклад](/docs/extensions/mysklad) | Каталог товаров, остатки, обработка заказов |

## Как подключить расширение

Каждое расширение подключается из раздела **Расширения** в панели управления Pleep. Процесс подключения зависит от конкретного сервиса — подробные инструкции доступны на странице каждого расширения.
