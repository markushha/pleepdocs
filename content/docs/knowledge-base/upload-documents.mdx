---

Загрузка документов — самый быстрый способ наполнить базу знаний вашего AI-агента. Система поддерживает множество форматов и автоматически обрабатывает содержимое с помощью интеллектуальной обработки.

## Поддерживаемые форматы

- **PDF** — прайс-листы, каталоги, презентации
- **DOCX / DOC** — документы Microsoft Word
- **XLS / XLSX** — таблицы Microsoft Excel
- **ODS** — таблицы OpenDocument
- **ODT** — документы OpenDocument
- **TXT** — текстовые файлы
- **JSON** — структурированные данные
- **RTF / RTFD** — форматированный текст

## Как загрузить документы

<Steps>

### Откройте базу знаний

Перейдите в раздел **База знаний** в панели управления вашего агента.

### Нажмите «Загрузить»

Нажмите кнопку загрузки файлов.

### Выберите файлы

Выберите один или несколько файлов с вашего компьютера. Можно загружать сразу несколько файлов одновременно.

### Дождитесь обработки

Система запустит **интеллектуальную обработку** — AI автоматически проанализирует содержимое, разобьёт его на логические блоки. Статус документа изменится с «обработка» на «индексирован».

### Проверьте результат

После обработки вы можете просмотреть, как система структурировала информацию. При необходимости отредактируйте отдельные блоки вручную.

</Steps>

## Интеллектуальная обработка

При загрузке документа система не просто извлекает текст — она использует AI для интеллектуального разбиения на логические элементы. Например:

- Прайс-лист будет разбит на отдельные товары с ценами
- Каталог будет разделён на категории и товары
- Текст будет разбит на тематические блоки

Это позволяет AI-агенту быстрее находить нужную информацию и давать более точные ответы.

## Управление документами

- **Статус** — отслеживайте состояние каждого файла: обработка, индексирован, ошибка
- **Переиндексация** — можно запустить повторную обработку отдельного документа без его повторной загрузки
- **Редактирование блоков** — вы можете создавать и редактировать отдельные блоки данных вручную

<Callout title="Google Docs и Sheets">
  Pleep поддерживает прямой импорт из Google Docs и Google Sheets. Однако данные не обновляются автоматически — после внесения изменений в документ Google его нужно импортировать заново.
</Callout>
