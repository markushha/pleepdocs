---
title: Сканирование сайта
description: Как автоматически импортировать контент с вашего сайта в базу знаний
---

Если у вас есть сайт с информацией о бизнесе, Pleep может автоматически отсканировать его и добавить содержимое в базу знаний AI-агента. HTML-страницы автоматически конвертируются в markdown.

## Два режима сканирования

| Режим | Описание |
| --- | --- |
| **Одна страница** | Сканирование только указанной страницы |
| **Обход сайта** | Автоматический обход нескольких страниц по внутренним ссылкам |

## Как запустить сканирование

<Steps>

### Откройте базу знаний

Перейдите в раздел **База знаний** в панели управления вашего агента.

### Выберите источник «Сайт»

Нажмите на **Сайт** среди доступных источников данных.

### Введите URL

Укажите адрес страницы, которую хотите отсканировать. Система проверит валидность URL перед началом сканирования.

### Выберите режим сканирования

Выберите сканирование одной страницы или обход нескольких страниц сайта.

### Запустите сканирование

Нажмите кнопку запуска. Система автоматически загрузит страницы и конвертирует HTML в markdown.

### Проверьте и отредактируйте контент

После сканирования вы можете просмотреть и отредактировать полученный контент перед сохранением в базу знаний.

</Steps>

## Автоматическая конвертация

Pleep автоматически конвертирует HTML-страницы в чистый markdown, убирая навигацию, футеры и другие служебные элементы. В базу знаний попадает только полезное текстовое содержимое.

<Callout title="Совет">
  Для максимального охвата информации используйте режим обхода сайта, начиная с главной страницы или раздела с основной бизнес-информацией.
</Callout>
