The knowledge base is the single source of truth for your AI agent. Every response the agent gives is grounded in the data you have added to the knowledge base. The more complete and high-quality it is, the more accurate and useful the responses will be.

## How It Works

The AI agent does not make up answers — it finds relevant information in your knowledge base and formulates a response in language the customer can understand. If the necessary information is not in the knowledge base, the agent will honestly say it cannot answer the question.

When you add data, the system uses **smart processing** — intelligent document splitting into logical blocks. This allows the agent to find the right information quickly and accurately.

## Data Sources

You can populate the knowledge base from several sources:

| Source | Description |
| --- | --- |
| **File upload** | PDF, DOCX, DOC, XLS, XLSX, ODS, ODT, TXT, JSON, RTF, RTFD |
| **Website scraping** | Single page or multi-page crawl, auto-converts to markdown |
| **Instagram scraping** | Import content from an Instagram account with AI filtering |
| **Manual entry** | Q&A pairs or free-form text |
| **Google Docs / Sheets** | Auto-fetch and import documents from Google |

## Content Management

After adding data, you can:

- **Track status** of each document: processing, indexed, error
- **Reprocess** individual documents without re-uploading
- **Create and edit blocks** manually for fine-tuning
- **Clear agent cache** to apply updates

## Where to Start

We recommend starting with your most important documents — price lists, product catalogs, and service descriptions. Then add information from your website and Instagram.

<Callout title="Important">
  Regularly update your knowledge base so the agent always provides up-to-date information. Outdated data can lead to inaccurate responses.
</Callout>
