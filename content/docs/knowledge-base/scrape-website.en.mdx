---
title: Scrape Website
description: How to automatically import content from your website into the knowledge base
---

If you have a website with business information, Pleep can automatically scrape it and add the content to your AI agent's knowledge base. HTML pages are automatically converted to markdown.

## Two Scraping Modes

| Mode | Description |
| --- | --- |
| **Single page** | Scrape only the specified page |
| **Multi-page crawl** | Automatically crawl multiple pages by following internal links |

## How to Start Scraping

<Steps>

### Open the Knowledge Base

Go to the **Knowledge Base** section in your agent's dashboard.

### Select "Website" as the source

Click on **Website** among the available data sources.

### Enter the URL

Enter the address of the page you want to scrape. The system will validate the URL before starting.

### Choose the scraping mode

Select single page scraping or multi-page crawl.

### Start the scrape

Click the start button. The system will automatically download the pages and convert HTML to markdown.

### Review and edit the content

After scraping, you can review and edit the resulting content before saving it to the knowledge base.

</Steps>

## Automatic Conversion

Pleep automatically converts HTML pages to clean markdown, removing navigation, footers, and other non-content elements. Only useful text content is added to the knowledge base.

<Callout title="Tip">
  For maximum information coverage, use multi-page crawl mode starting from your home page or the section with your main business information.
</Callout>
