## What should I add to the knowledge base?

Upload **all information about your business** that helps sell and serve customers:

- Products and services — descriptions, specifications
- Prices and price lists
- Promotions and special offers
- Delivery and payment terms
- Contact information — addresses, phone numbers, business hours
- Return and warranty policies
- Answers to typical customer questions

The more information you add, the better the agent will respond.

## What is smart processing?

Smart processing is AI-powered document processing that automatically splits uploaded files into logical elements. For example:

- A price list will be broken down into individual products with prices
- A catalog will be divided into categories and products
- Text will be split into thematic blocks

This allows the AI agent to find the right information faster and provide more accurate answers.

## Can I add Google Docs and Sheets?

Yes, Pleep supports direct import from Google Docs and Google Sheets. However, data **does not update automatically** — after making changes to a Google document, you need to re-import it into the knowledge base.

## How do I update content in the knowledge base?

There are several ways:

- **Upload a new version** of the document — it will replace the old one
- **Edit blocks manually** — open the document in the knowledge base and modify individual data blocks
- **Create a new entry** via manual entry to add missing information

## What happens during document reprocessing?

During reprocessing, the document goes through smart processing again: the system re-splits the content into logical elements. This is useful when you want to update a document's processing without re-uploading the file.

## Can I delete uploaded data?

Yes, you can delete any item from the knowledge base at any time. After deletion, the agent will stop using that information in its responses.

## What if the agent gives inaccurate answers?

Pay attention to the agent's responses in real conversations. If the agent:

- Gives inaccurate or outdated answers — update the relevant information
- Cannot answer a question — add the necessary data
- Responds too generically — add more detailed information

Also try **clearing the agent cache** after updating the knowledge base so changes take effect faster.
