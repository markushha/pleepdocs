База знаний — это единственный источник правды для вашего AI-агента. Каждый ответ агента основан на данных, которые вы добавили в базу знаний. Чем полнее и качественнее она заполнена, тем точнее и полезнее будут ответы клиентам.

## Как это работает

AI-агент не придумывает ответы — он находит релевантную информацию в вашей базе знаний и формулирует ответ на понятном для клиента языке. Если нужной информации в базе нет, агент честно скажет, что не может ответить на вопрос.

При добавлении данных система использует **интеллектуальную обработку** — интеллектуальное разбиение документов на логические блоки. Это позволяет агенту быстро и точно находить нужную информацию.

## Источники данных

Вы можете наполнять базу знаний из нескольких источников:

| Источник | Описание |
| --- | --- |
| **Загрузка файлов** | PDF, DOCX, DOC, XLS, XLSX, ODS, ODT, TXT, JSON, RTF, RTFD |
| **Сканирование сайта** | Одна страница или полный обход сайта, автоматическая конвертация в markdown |
| **Сканирование Instagram** | Импорт контента из Instagram-аккаунта с AI-фильтрацией |
| **Ручной ввод** | Пары вопрос-ответ или свободный текст |
| **Google Docs / Sheets** | Автоматическое получение и импорт документов из Google |

## Управление контентом

После добавления данных вы можете:

- **Отслеживать статус** каждого документа: обработка, индексирован, ошибка
- **Переиндексировать** отдельные документы без повторной загрузки
- **Создавать и редактировать блоки** вручную для тонкой настройки
- **Очистить кэш агента** для применения обновлений

## С чего начать

Рекомендуем начать с загрузки самых важных документов — прайс-листов, каталогов товаров, описаний услуг. Затем добавьте информацию с сайта и Instagram.

<Callout title="Важно">
  Регулярно обновляйте базу знаний, чтобы агент всегда давал актуальную информацию. Устаревшие данные могут привести к неточным ответам.
</Callout>
