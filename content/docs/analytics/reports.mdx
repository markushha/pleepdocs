Раздел отчётов предоставляет более детальный анализ работы вашего AI-агента с возможностью выбора периода и разбивкой по каналам.

## Доступные отчёты

### Разбивка по каналам

Для каждого подключённого канала (WhatsApp, Instagram, Telegram и др.) вы можете увидеть:

- Количество обработанных сообщений.
- Долю AI-ответов и ответов операторов.
- Динамику обращений за выбранный период.

### Отчёт за период

Выберите нужный диапазон дат для просмотра:

- **Почасовые данные** — для анализа пиковых часов нагрузки.
- **Ежедневные данные** — для отслеживания динамики по дням.
- **Ежемесячные данные** — для стратегического планирования.

### Потребление сообщений

Отчёт показывает:

- Общий расход сообщений за период.
- Распределение между AI и оператором.
- Остаток лимита сообщений.

## Интеграция с Google Analytics

Вы можете подключить Google Analytics для расширенного отслеживания:

1. Перейдите в настройки приложения.
2. Найдите раздел **Google Analytics**.
3. Введите ваш идентификатор отслеживания (Measurement ID).
4. Сохраните настройки.

После подключения данные о взаимодействиях с AI-агентом будут отправляться в вашу панель Google Analytics, что позволяет объединить аналитику чат-бота с общей аналитикой вашего бизнеса.
