title: First Steps
description: How to test and configure your AI agent after creation
---

Your AI agent has been created. Now let's make sure it works as expected and prepare it for real customer conversations.

<Steps>

### Test in the Sandbox

The Sandbox is a built-in chat where you can interact with your AI agent just like your customers will.

- Send test messages — ask typical customer questions
- Check how accurately and convincingly the agent responds
- Try objections — "too expensive", "I'll think about it", "competitors are cheaper"

<Callout title="Tip">
  Try asking questions the agent might not have an answer to. This will help you understand what information needs to be added to the Knowledge Base.
</Callout>

### Improve responses

If you are not satisfied with the agent's responses — open the **Improve** page and describe in plain language what needs to change. The AI will automatically update the sales script based on your instructions.

### Connect a messenger

When you are satisfied with the agent's performance, connect it to communication channels:

- **WhatsApp** — via WhatsApp Business API
- **Instagram** — via Instagram Direct
- **Telegram** — via Telegram Bot
- **Website widget** — an embeddable chat for your website

Once connected, the AI agent will automatically start responding to customers.

### Set up notifications

Receive alerts in Telegram when:

- A new customer conversation starts
- A customer asks to speak with a live agent
- The AI determines that a customer is ready to buy

</Steps>
